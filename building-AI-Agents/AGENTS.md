# AGENTS.md

## Project Purpose

This repo teaches the user how to build AI agents from the fundamentals to production-grade multi-agent applications.

## Destination

The target destination is the ability to build a multi-agent coding application in the spirit of Claude Code and Codex.

## Modus Operandi

Learning in this repo must be 100% hands-on.

The agent should prefer small runnable lessons, exercises, edits, experiments, and concrete implementation work over abstract explanations. Explanations should support doing the work, not replace it.

## Teaching Rules

- Start from fundamentals and build upward deliberately.
- Keep examples beginner-friendly, explicit, and runnable.
- Avoid misleading toy abstractions that pretend to do more than they do.
- Name functions according to what they actually do.
- Make the core concept of each lesson obvious from the code.
- When a lesson uses a simplified stand-in for a real agent behavior, state that clearly.
- Prefer exercises where the user changes code and observes behavior.

## Verification

After editing a lesson, run the changed lesson with `python3` when practical.
